# Espaço Acolher - API Backend

Backend API para o sistema de agendamentos da clínica Espaço Acolher.

## Deploy no Render

Este repositório contém apenas o código compilado pronto para produção.

### Configuração:

**Build Command:** `npm install`

**Start Command:** `npm start`

### Variáveis de Ambiente:

- `DATABASE_URL`: URL de conexão PostgreSQL
- `NODE_ENV`: production
- `PORT`: 10000

## Funcionalidades

- API REST para sincronização de dados
- Gerenciamento de pacientes, profissionais, especialidades e agendamentos
- Suporte a modo offline com sincronização automática
